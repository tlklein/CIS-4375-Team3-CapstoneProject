# BACKEND INSTRUCTIONS  

1. Open a terminal and navigate to the backend directory using Visual Studio Code or Command Prompt
2. npm install
3. npm run dev (development)
4. npm start (production)
5. Create a .env file in the backend directory
6. Copy and paste the following into the .env file:

    ```env
    DB_HOST=site.0123456789.us-east-1.rds.amazonaws.com

    DB_USER=admin

    DB_PASSWORD=password

    DB_NAME='DBName'

    PORT=3004
    ```

7. To start backend, first log in to AWS Academy
8. Click "Modules" and click "Start Lab"
9. Wait for the dot to change from red to green
10. Click "AWS" at the top to open the console
11. In the AWS Console, go to "RDS"
12. Locate your database instance and click "Start" to launch it
