# Innov8 Barber Shop Management System

![banner](/Documentation/Screenshot%202025-04-09%20153009.png)

## Overview

The Barber Shop Management System will be a web-based solution
enabling Carlos Zuniga, a local barber, to manage appointments,
customer information, and basic business data.

The goal is to streamline daily operations, reduce scheduling
conflicts, and enhance customer service. This project combined
full-stack development with applied project management,
emphasizing requirements gathering, system design, and iterative
delivery.
